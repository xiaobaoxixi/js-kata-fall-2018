Array methods

These are some of the more useful array methods

Info-methods

These methods gives some information about the array

  • indexOf
  • find
  • findIndexOf
  • every
  • includes

Modifying methods

These methods modifies the array directly

  • concat
  • copyWithin
  • fill
  • join
  • sort
  • splice

New Array

These methods gives a new array, based on the original

  • filter
  • map
  • reduce
  • slice
